# 老虎機遊戲

一個簡單有趣的老虎機遊戲。

## 功能特點

- 現代化的UI設計
- 流暢的動畫效果
- 積分系統
- 響應式設計，支持移動端
- 加速下注功能

## 如何運行

1. 克隆此倉庫
2. 使用瀏覽器打開 `index.html` 文件
3. 開始遊戲！

## 遊戲規則

1. 點擊"開始"按鈕開始遊戲
2. 三個相同的符號可以獲得獎勵
3. 特殊符號組合可以獲得額外獎勵
4. 使用積分進行下注
5. 可以選擇1-10次連續下注

## 獎勵倍數

- 💎 (鑽石): 10倍
- 7️⃣ (7): 8倍
- 🎰 (老虎機): 6倍
- 水果符號: 3倍

## 技術棧

- HTML5
- CSS3
- JavaScript (ES6+)
